# CCT-Demo

Claims Centre of Truth (CCT) is a fraud prevention and investigation platform for insurers. This repo contains the React/Vite web client with data services, risk insights, and investigations workflows.

## Goals

- Present claim, device, and case data through a unified dashboard.
- Enable investigators to triage high-risk devices and manage cases with audit logging.
- Demonstrate role-based controls for analysts and managers within a demo-grade UI.
